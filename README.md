##  Featured Project

### 🫁 Pneumonia Detection from Chest X-Rays

**Repository:** [pneumonia-detection-ai](https://github.com/RaouaNouioua/pneumonia-detection-ai)

End-to-end medical AI system for binary classification of chest X-rays into NORMAL vs PNEUMONIA categories. Complete pipeline from data preprocessing to production deployment with 86.1% accuracy and clinically-validated safety metrics.

**Technical Implementation:**

- PyTorch + ResNet18 architecture with transfer learning from ImageNet
- FastAPI backend serving real-time predictions (<2 seconds)
- React frontend with drag-drop interface and Grad-CAM visualizations
- Class imbalance handling using weighted loss function (3:1 ratio)
- Model explainability through Grad-CAM heatmaps for radiologist validation

**Problem Solved:** 
Reduces false negatives in pneumonia detection to <3% (critical for patient safety) while cutting false positives by 50% compared to baseline models. Transforms manual X-ray review into AI-assisted diagnosis with clinical recommendations.

**Key Innovation:** 
First implementation combining transfer learning (86.1% accuracy) with medical-grade safety metrics - prioritizing recall (97.4%) over precision to ensure no pneumonia case is missed. Includes explainability layer showing exactly where the model detects lung opacities.
